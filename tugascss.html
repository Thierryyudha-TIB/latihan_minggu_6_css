TUGAS CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk menentukan tampilan dan tata letak elemen dalam sebuah halaman web. CSS bekerja bersama dengan HTML untuk mengatur warna, font, ukuran, posisi, dan berbagai aspek desain lainnya agar halaman web lebih menarik dan responsif.

HTML CSS
Struktur dan konten halaman web. Mengatur tampilan dan desain halaman web.
Menyediakan elemen-elemen seperti teks, gambar, tabel, formulir, dll. Mengubah warna, font, layout, dan efek visual.
Dibaca langsung oleh browser untuk menampilkan konten. Digunakan bersama HTML untuk memperindah tampilan.
